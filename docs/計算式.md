# 計算式

## 目的

この文書は、`v0.2` 以降で目指す **日本全域の連続的な VTEC 場推定** の計算方針を定義する。

`v0.1` では `IPP -> cell aggregate` を主としていたが、`v0.2` ではそれを最終形としない。  
今後は **全国格子の VTEC を未知数として、全 LOS 観測から同時に推定する**。

## 位置づけ

### `v0.1` で残すもの

- `ringo qc`
- `ringo rnxcsv`
- 観測コード選択
- geometry-free combination
- cycle slip 検出
- arc segmentation
- station metadata
- IPP 計算の補助

### `v0.2` で見直すもの

- GPS 優先から multi-GNSS へ拡張
- LLI / SNR / elevation を使った品質管理
- receiver / satellite / arc bias の明示的推定
- `IPP -> cell aggregate` から `regional VTEC field estimation` へ変更
- quiet-day baseline の導入

## 入力

### 観測データ

- RINEX observation files
- navigation files
- station metadata

### 中間入力

`ringo rnxcsv` の出力を使う。

対象は将来的に次を含む。

- GPS
- GLONASS
- Galileo
- QZSS

## 観測モデル

GNSS 搬送波位相とコード観測は、簡略化すると次で表す。

```math
\Phi_i =
\rho +
c(\delta t_r - \delta t_s) +
T -
I_i +
\lambda_i N_i +
b_{\Phi_i,r} +
b_{\Phi_i,s} +
\epsilon_{\Phi_i}
```

```math
P_i =
\rho +
c(\delta t_r - \delta t_s) +
T +
I_i +
b_{P_i,r} +
b_{P_i,s} +
\epsilon_{P_i}
```

記号:

- $\rho$: 幾何学距離
- $c(\delta t_r - \delta t_s)$: 受信機・衛星時計差
- $T$: 対流圏遅延
- $I_i$: 電離圏遅延
- $\lambda_i N_i$: 位相アンビギュイティ
- $b$: 受信機・衛星・観測系のバイアス

一次電離圏項は周波数二乗に反比例する。

```math
I_i \propto \frac{\mathrm{STEC}}{f_i^2}
```

## Step 1: multi-GNSS observation selection

同一衛星・同一時刻に複数コードがある場合、周波数ごとに優先順位を設けて 1 系列に落とす。

`v0.2` では constellation ごとに独立の優先順位を持つ。

例:

- GPS: `L1`, `L2`, `C1`, `C2`
- Galileo: `E1`, `E5`
- QZSS: `L1`, `L2`, `L5`

## Step 2: quality control

全国場推定に入れる前に観測を厳しく落とす。

基本条件:

```math
E < E_{\min} \Rightarrow \text{discard}
```

```math
\mathrm{SNR} < S_{\min} \Rightarrow \text{discard}
```

```math
\mathrm{LLI} \neq 0 \Rightarrow \text{slip candidate}
```

ここで:

- $E_{\min}$: 仰角マスク
- $S_{\min}$: SNR 閾値

## Step 3: geometry-free combination

二周波観測から geometry-free combination を作る。

```math
L_{\mathrm{GF}} = \lambda_1 L_1 - \lambda_2 L_2
```

```math
P_{\mathrm{GF}} = P_2 - P_1
```

これにより、幾何距離・時計差・対流圏の共通成分を落とし、電離圏項を強調する。

## Step 4: cycle slip detection and arc segmentation

arc 分割は引き続き必要である。

基本条件:

```math
\Delta t > t_{\mathrm{gap}}
```

```math
|\Delta L_{\mathrm{GF}}| > \tau_{\mathrm{GF}}
```

```math
\mathrm{LLI} \neq 0
```

将来的には Melbourne-Wubbena なども追加候補とする。

## Step 5: STEC estimation

二周波の geometry-free から STEC を導く。

```math
\mathrm{STEC}_{\Phi}
=
\frac{f_1^2 f_2^2}{40.3 \times 10^{16}(f_1^2 - f_2^2)}
(\lambda_1 L_1 - \lambda_2 L_2)
```

```math
\mathrm{STEC}_{P}
=
\frac{f_1^2 f_2^2}{40.3 \times 10^{16}(f_1^2 - f_2^2)}
(P_2 - P_1)
```

`v0.1` の code leveling は、`v0.2` では bias 推定の初期値として扱う。

## Step 6: bias estimation

`v0.2` では bias を明示的に扱う。

最終的に使いたい形は次である。

```math
\mathrm{STEC}_{\mathrm{obs}}
=
\mathrm{STEC}_{\mathrm{true}} +
B_r +
B_s +
B_{\mathrm{arc}} +
\epsilon
```

ここで:

- $B_r$: receiver differential code bias
- $B_s$: satellite differential code bias
- $B_{\mathrm{arc}}$: arc ごとの位相オフセット

推定方針:

- code 観測で absolute 側を拘束
- phase 観測で低ノイズ系列を確保
- arc bias を未知数として同時推定

## Step 7: IPP and mapping function

thin-shell model は引き続き使う。

```math
H = 350 \ \mathrm{km}
```

mapping function:

```math
M(E) =
\frac{1}{\sqrt{
1 -
\left(
\frac{R_E \cos E}{R_E + H}
\right)^2
}}
```

```math
\mathrm{VTEC} = \frac{\mathrm{STEC}}{M(E)}
```

ただし `v0.2` では、`IPP に置いた値をそのまま格子平均する` のではなく、**観測方程式を全国格子の未知数へ結びつける補助量**として IPP を使う。

## Step 8: regional VTEC field estimation

ここが `v0.2` の中心である。

### Unknowns

時刻 $t$ ごとの全国格子 VTEC を未知数ベクトル $\mathbf{x}_t$ とする。

```math
\mathbf{x}_t =
[VTEC_{1,t}, VTEC_{2,t}, \dots, VTEC_{n,t}]^\top
```

### Observation equation

各 LOS 観測 $k$ に対して、

```math
\mathrm{STEC}_k
=
M(E_k)
\sum_{j=1}^{n}
w_{k,j} \,
VTEC_{j,t}
+
B_{r(k)}
+
B_{s(k)}
+
B_{\mathrm{arc}(k)}
+
\epsilon_k
```

ここで:

- $w_{k,j}$: IPP 近傍格子への補間重み
- $M(E_k)$: mapping function

行列表現:

```math
\mathbf{y} = \mathbf{A}\mathbf{x} + \mathbf{G}\mathbf{b} + \boldsymbol{\epsilon}
```

- $\mathbf{y}$: LOS 観測ベクトル
- $\mathbf{x}$: 全国格子 VTEC
- $\mathbf{b}$: bias ベクトル

### Regularization

格子をそのまま解くと不安定なので、空間・時間正則化を入れる。

```math
\min_{\mathbf{x}, \mathbf{b}}
\left\|
\mathbf{y} - \mathbf{A}\mathbf{x} - \mathbf{G}\mathbf{b}
\right\|^2
+
\lambda_s \left\| \mathbf{L}_s \mathbf{x} \right\|^2
+
\lambda_t \left\| \mathbf{L}_t \mathbf{x} \right\|^2
```

ここで:

- $\mathbf{L}_s$: 空間平滑化
- $\mathbf{L}_t$: 時間平滑化
- $\lambda_s, \lambda_t$: 正則化係数

これにより、観測の無い海上や周辺部にも連続的な場を与える。

## Step 9: baseline and anomaly

異常抽出は quiet-day baseline に寄せる。

同一 local time に対して:

```math
B(t_{\mathrm{LT}}) = \mathrm{median}\left(\mathrm{VTEC}_{\mathrm{quiet}}(t_{\mathrm{LT}})\right)
```

```math
\sigma(t_{\mathrm{LT}}) = \mathrm{std}\left(\mathrm{VTEC}_{\mathrm{quiet}}(t_{\mathrm{LT}})\right)
```

```math
z(t) = \frac{\mathrm{VTEC}(t) - B(t_{\mathrm{LT}})}{\sigma(t_{\mathrm{LT}})}
```

必要なら detrended TEC も別に保持する。
